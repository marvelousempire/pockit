# **Feature Ledger – Detailed Breakdown**

**1. Hybrid Value System**  
Captures two separate signals for every item or activity: what the user *says* they value (explicit rating) and what their actual behavior shows (observed actions). The system tracks the gap between the two to understand true priorities.

**2. Ownership Fracturing**  
Users can split off any percentage of their own ownership or equity and assign it to others (investors, partners, contributors) without reducing anyone else’s share. Full history of every fracture is tracked.

**3. Node Relationship Mapping**  
A visual graph system that shows how people, ideas, investments, and value flow between each other. Every new connection or contribution creates a traceable node and branch.

**4. 0.0–1.0 Precision Rating Scale**  
All ratings in the app use a 0.0 to 1.0 scale. 1.0 represents an “Unwavering Yes” — the highest possible rating. This forces higher intentionality when users rate anything.

**5. Color Wheel Mood Input**  
A large, continuous color spectrum interface where users drag their finger to select exactly how they feel. Each color maps to emotional and energy data behind the scenes.

**6. Micro-Question Engine**  
The core data collection system. It sends short, frequent questions using specific time windows (example: “In the last 2 hours, how many times did you…?” with 0 / 1 / 2-3 / 4+ options). Designed to feel random but actually strategic.

**7. Consistency & Validation Engine**  
When a user gives an extreme rating (especially 1.0), the system later re-asks similar questions in different forms to test consistency. It quietly builds a reliability score for each user based on how consistent their answers are over time.

**8. Voice Journal Agent**  
A proactive AI agent that triggers voice-based journal entries throughout the day. It starts natural conversations like “How was basketball earlier?” The user speaks freely, the app records, transcribes, summarizes, and extracts structured data from the conversation. Creates a complete daily journal automatically.

**9. Personal Data Absorption Agent**  
The central AI brain of the app. Its only purpose is to continuously absorb truthful data about the user through all the above systems to build the most accurate understanding of who the user really is.

**10. Pattern Recognition & Insight Engine**  
Analyzes everything collected to find real patterns — mood cycles, productivity gaps, contradictions between words and actions — and surfaces honest insights to the user.

**11. Long-term Memory Bank**  
All data (ratings, journal entries, mood inputs, micro-question answers) is stored in a structured, searchable long-term memory system that feeds the user’s personal AI agent over time.

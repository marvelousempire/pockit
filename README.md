# README and Guide

-----

Pockit — Your Fractictioner AI Secretary

Pockit is a personal truth engine that builds a deeply accurate AI agent by quietly learning who you really are through gentle daily interactions.

### Core Philosophy

Pockit respects your time and earns your trust. It gathers honest behavioral and emotional data while delivering real value from day one. Your data never leaves your control.

### Core Experience

- **Color Wheel Mood Input** — Beautiful continuous spectrum for instinctive mood tracking
- **0.0–1.0 Precision Ratings** — Honest sliding scale where 1.0 means “Unwavering Yes”
- **Intelligent Micro-Questions** — Short, contextual questions delivered at smart moments
- **Voice Journal Agent** — Speak naturally and let Pockit capture your thoughts
- **Voice Memo Input** — Quick voice responses when you prefer to talk

### Voice Processing

Built with **Whisper.cpp + Silero VAD** for fast, accurate, and fully self-hosted speech recognition. Optimized to run efficiently on edge devices like Raspberry Pi and Mac Mini.

### Intelligence Layer

- Basic Long-term Memory Bank
- Simple Pattern Recognition & Insights
- Light Consistency Engine that quietly validates your responses over time

### Architecture & Security

- **Self-hosted by design** — Runs on your own VPS, Mac Mini, or server
- Requires valid HTTPS certificate
- All personal data stays on your infrastructure
- Full data export and deletion capabilities

### Tech Stack (MVP)

- **Backend**: FastAPI
- **Speech**: Whisper.cpp + Silero VAD
- **Frontend**: React Native / Flutter
- **Deployment**: Docker + docker-compose

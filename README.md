# Pockit — universal console platform

Pockit is the **frame**. Your consoles and cassettes are **products** you hang on the frame.

This repo ships the platform shell — rails, encompass, viewport tiers (watch → 85″ TV), help console UI, intent model, changelog law, accessory desk — plus a **demo catalog** (one player, three sample cassettes). It does **not** ship the Family Office fleet (Voice, SME archive, WordPress, 134 cassettes).

## Quick start

```bash
git clone ssh://git@10.1.0.5:2424/marvelousempire/pockit.git
cd pockit
cat AGENTS.md          # AI build law — read first
npm test               # boundary check (no product IP in shell/)
```

Operator runtime (full stack): `cd ~/Developer/nephew && make pockit` → http://pockit.localhost/

Refresh shell after nephew changes: `make pockit-saas-export`

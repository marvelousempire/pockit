### Phase 1: MVP (Target: 4–6 weeks)

**Goal**: A minimum lovable product that delivers real value quickly.

**Key Deliverables**

- Full project setup (Docker, FastAPI skeleton, folder structure)
- Voice Pipeline: Whisper.cpp + Silero VAD (quantized models)
- Core Input System: Color Wheel + 0.0–1.0 ratings
- Micro-Question Engine (basic scheduler + generator)
- Basic Memory Layer with Matryoshka embeddings
- Simple Dashboard with early insights
- One-command `setup.sh` for easy self-hosting

**Success Criteria**

- Onboarding in < 3 minutes
- Users get at least 3 meaningful insights within 10 days
- Runs well on Raspberry Pi 5 and Mac Mini

**Target Release**: v0.1 (End of June 2026)
